## 🚀 快速开始

### 1. 设置虚拟环境（推荐）

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境（macOS/Linux）
source venv/bin/activate

# 激活虚拟环境（Windows）
venv\Scripts\activate
```

### 2. 运行示例代码

```bash
# 进入对应目录
cd day1

# 运行 Python 文件
python3 hello_python.py
```

## 📝 学习记录

## 🎯 学习建议

1. **每天按计划学习**：按照 `PYTHON学习计划.md` 中的安排逐步学习
2. **多写代码**：光看不练假把式，一定要多写代码
3. **对比学习**：利用前端经验，对比 JavaScript 和 Python
4. **做笔记**：记录重要概念和 Python 特有的语法
5. **项目驱动**：通过实际项目巩固知识
